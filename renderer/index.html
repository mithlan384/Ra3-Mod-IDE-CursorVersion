欢迎使用 RA3 MOD IDE

面向《命令与征服：红色警戒 3》模组开发的集成环境 — 集中编辑 XML、预览资源、调用 SDK 编译，并用自然语言与 AI 协作完成 MOD 制作。

当前项目

本 IDE 面向《红色警戒 3》MOD 制作全流程：在左侧文件树编辑 XML、预览 TGA/DDS/W3X/CSF，用 Ctrl+B 调用 SDK 后台编译，通过状态栏 AI 扫描项目、创建单位、修复 BAE 报错。内置 knowledge-docs 教程库与混合检索，减少翻论坛找标签的时间。

近期能力包括：文件名模糊搜索、AI 写入时界面锁定、项目定时完整备份与菜单恢复、路径沙箱防越界改文件、T1 权限下只读工具免确认、会话按项目记忆打开的标签与 AI 对话等。下方每张功能卡片均可点开阅读完整说明（含操作步骤、快捷键与注意事项）。

首次使用：菜单 文件 → 打开项目 或项目管理器选择 MOD；在「设置 → 首选项」配置 RA3 MOD SDK 路径与（可选）AI API Key。菜单 帮助 → 关于 RA3 IDE 可随时回到本页。
