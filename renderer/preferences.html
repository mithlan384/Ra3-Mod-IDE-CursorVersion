自动保存

仅保存已打开且未保存的文件到当前 MOD 目录，不是整包备份。
关闭后重新打开时恢复本次设置

项目自动备份

按周期将当前打开的 MOD 根目录复制到备份文件夹；备份前会尽量保存已修改的打开文件。
例如 0 天 1 小时 30 分 = 每 90 分钟备份一次。天、时、分不能同时为 0。
超出上限后删除最早的备份。文件夹命名：项目名-备份-时间戳。

SDK 设置

选择 RA3 MOD SDK-X 的根目录（包含 EALAModStudio.exe 的文件夹）

外部工具路径（可选，留空则自动在 SDK/tools 下查找）

设置 SDK 根目录后可自动填充；文件树右键与预览页可启动对应工具。

AI 设置

与 AI 对话中「标准/当前项目格式」选择联动。选「当前项目」且尚未学习时，保存后将在 AI 面板提示扫描确认。
仅在选择 T1 或 T2 权限时生效；T3 完全自主无需确认。
每「步」为 Agent 向大模型请求一轮（一轮内可执行多个工具）。复杂 MOD 任务建议 48 或无限制；对话进行中可点发送栏「停止」终止。

联网搜索设置

「自动」无需 API Key，会通过 DuckDuckGo / 公共 SearXNG 真实联网
请确保 SearXNG 实例支持 JSON API (format=json)
