RA3 MOD IDE — 使用说明（v0.4）

================================



【无需 Node.js】本安装包已内置运行环境，无需命令行或 npm。



一、推荐方式（解压即用）

------------------------

1. 将「RA3-MOD-IDE-0.4.zip」解压到任意目录（路径勿含奇怪符号）

2. 进入解压后的「RA3 MOD IDE」文件夹

3. 双击「启动 RA3 MOD IDE.bat」或「RA3 MOD IDE.exe」



也可直接使用已解压好的「RA3 MOD IDE」文件夹（无需再解压 zip）。



二、安装程序（可选）

--------------------

若同目录有「RA3 MOD IDE-0.4-Setup.exe」，可按向导安装到本机。



三、首次使用

------------

1. 启动后出现「项目管理」窗口，选择或浏览 MOD 项目文件夹

2. 在本机配置 RA3 MOD SDK 路径（欢迎页或 设置 → 首选项 → SDK 设置）

   ※ 本包不包含 SDK，需自行安装 EA 官方 RA3 MOD SDK-X

3. 在首选项 → AI 设置 填写 DeepSeek（或其它兼容）API Key 后即可使用 AI



四、界面素材说明

----------------

安装包已内置 Art 美术资源（项目管理器壁纸、阵营壁纸幻灯片、Logo、头像等），

位于安装目录 resources\Art\，启动时自动加载。

可在 设置 → 首选项 → 主题 切换 Dark / Light / 高对比度 / 盟军 / 苏联 / 帝国 验证显示是否正常。

若壁纸或 Logo 不显示，请反馈安装方式与 Windows 版本。



五、与开发版的区别

------------------

本包已清除：开发者 SDK 路径、最近项目列表、编辑器会话、AI 对话记录、API Key。

首次启动写入默认首选项，其余由测试员在本机配置。



六、数据位置

------------

用户配置与对话保存在：

%APPDATA%\ra3-ide\



七、反馈请提供

--------------

Windows 版本、安装版/绿色版、操作步骤、截图（含主题/壁纸是否正常）。
