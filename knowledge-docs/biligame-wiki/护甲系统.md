# 护甲系统

> 来源：[红警3 B站百科 · 护甲系统](https://wiki.biligame.com/redalert3/%E6%8A%A4%E7%94%B2%E7%B3%BB%E7%BB%9F)
> 整理时间：2026-05-22

与星际争霸等 RTS 不同，RA3 **不分轻甲/重甲**，而是用「护甲类型 × 伤害类型 → 修正比（%）」的矩阵。修正比即伤害倍数：200% = 双倍伤害，50% = 减半。

**伤害类型（列）**：肉搏、狙击、枪弹、机炮、破片、火箭、穿甲、光谱、电击、高爆、榴弹、鱼雷、冲击、毒素（部分护甲另有「魔法」列）。

下表与百科原页一致（1.12）。MOD 改 `ArmorSet` / `DamageType` 时可对照。
